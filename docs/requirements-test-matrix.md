- 修改第一版需求、API、决策状态、Admin 边界、部署或测试门禁时，同步更新对应行的案例和证据。
- “自动化已覆盖”只表示仓库内存在可执行断言；完成状态仍需以本次验证命令或 CI 结果为准。
- live smoke 不打印密码、Cookie、Authorization、数据库 URL、Secret 值或原始客户数据，只记录状态、`decision_id` 与脱敏摘要。
